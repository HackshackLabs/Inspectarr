{% if sonarr_enabled %}

Sonarr column loads monitored state and Sonarr file count (episode files on disk) per row. Rows tint red when Sonarr reports the series, season, or episode as missing, or when there are no episode files on disk for that row’s scope. Use ⓘ for path details. Matching uses TVDB id when present, otherwise the series title against your Sonarr library. The three buttons are not duplicates: Unmonitor only changes monitoring and keeps files; Unmonitor + delete files unmonitors then deletes files (series or season stays in Sonarr); Remove series / Delete files only / Delete file only follow the third Sonarr path (drop series from Sonarr, or delete files without the unmonitor step). Full API mapping is documented in the repo file docs/SONARR.md. On per-server tables, when Plex is configured in Settings and the row has a ratingKey, the second and third buttons also run Plex to remove the same library item after Sonarr succeeds. Cumulative tables are Sonarr-only (no per-server rating key).
